# Examples 目录说明

## 当前状态

本目录暂未收录完整的示例文档。

## 原因

示例文档需要包含实际材料生成的真实内容，不应使用虚构数据占位。
为保持示例的参考价值，不在此放置无实际内容的空白模板。

## 参考资源

- 使用流程参考：`templates/usage-examples.md`（遗留层，含步骤示例）
- 输入模板参考：`quick-prompts/` 目录下各文档类型的结构化输入模板
- 完整模板参考：`document-types/` 目录下各文档类型的 `template.md`

## 如何贡献示例

如需添加真实示例：
1. 将材料去敏感化（替换为占位符）
2. 放置于本目录，文件命名格式：`[文档类型]-example-[序号].md`
3. 更新本 README 文件列表
