# 专题会议记录校验清单

生成文档后，逐项核查以下内容：

## 一、结构完整性

- [ ] 包含"会议基本情况"段落
- [ ] 包含"上阶段工作完成情况"段落
- [ ] 包含"存在的问题与分析"段落
- [ ] 包含"学习传达与政策要求"段落
- [ ] 包含"本阶段工作安排"段落
- [ ] 包含"会议要求"段落
- [ ] 后续落实事项有明确责任人和时限

## 二、内容准确性

- [ ] 会议时间、地点、参会人员信息准确
- [ ] 上阶段工作情况与信息源材料一致
- [ ] 工作安排与考核细则/通知要求对应
- [ ] 后续待办事项与会议讨论内容一致
- [ ] 无凭空编造的数据或事实

## 三、政治表述

- [ ] 学习传达内容有来源材料支撑
- [ ] 政治性语言表述准确，无过时引用
- [ ] "四个意识""两个维护"等规范表述使用正确

## 四、格式规范

- [ ] 标题层级符合规范
- [ ] 段落编号格式一致（"一、"和"（一）"层级正确）
- [ ] 无 Markdown 格式符号残留
- [ ] 后续落实事项表格格式清晰

## 五、信息缺口

- [ ] 所有"待补"标记已处理，或已告知用户需补充的内容
- [ ] 无以占位符代替实质内容的段落（[XXX] 仅用于真实空缺）
