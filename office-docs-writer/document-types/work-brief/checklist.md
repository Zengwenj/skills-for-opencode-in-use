# 工作简报校验清单

## 一、结构完整性

- [ ] 包含"基本情况"段落（背景简明）
- [ ] 包含"当前进展"段落（有时间节点和数据）
- [ ] 包含"重点信息"段落（有决策参考价值）
- [ ] 包含"下一步安排"段落（有时限和要求）

## 二、内容准确性

- [ ] 进展数据与信息源材料一致
- [ ] 时间节点准确
- [ ] 重点信息真实，无虚报

## 三、格式规范

- [ ] 简报标题规范（"关于...情况的简报"）
- [ ] 期次或日期标注正确
- [ ] 段落简洁，不超必要篇幅
- [ ] 落款格式正确

## 四、信息缺口

- [ ] "待补"标记均已处理
