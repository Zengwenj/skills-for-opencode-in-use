# 学习记录校验清单

生成文档后，逐项核查：

## 一、结构完整性

- [ ] 包含"学习基本情况"段落（时间、地点、参学人员）
- [ ] 包含"学习材料情况"段落，材料有来源说明
- [ ] 包含"学习体会和认识"段落，有实质内容
- [ ] 包含"贯彻落实措施"段落，措施具体可执行
- [ ] 持续关注事项段落已填写或已标注待补

## 二、内容准确性

- [ ] 学习材料名称准确，来源有据
- [ ] 学习体会结合了实际业务，不空泛
- [ ] 贯彻落实措施与学习内容逻辑一致
- [ ] 无凭空编造的学习内容或体会

## 三、政治表述

- [ ] 引用政治概念准确（"四个意识"等不遗漏、不错用）
- [ ] 学习内容对应当前有效的会议/文件精神
- [ ] 无过时或不当的政治表述

## 四、格式规范

- [ ] 标题与段落层级清晰
- [ ] 体会与措施均采用分条叙述，编号一致
- [ ] 无 Markdown 格式符号残留

## 五、信息缺口

- [ ] "待补"标记均已说明需要哪些信息
- [ ] 参学人员、主持人等关键信息均已填写或明确标注待补
