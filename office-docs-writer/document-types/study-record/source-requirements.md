# 学习记录信息源要求

## 必要信息

| 信息项 | 用途 |
|-------|------|
| 学习时间、地点、参学人员 | 文档头部基本信息 |
| 学习材料清单及来源 | 第二部分 |
| 学习材料内容（或摘要） | 体会与认识的依据 |

## 辅助信息

| 信息项 | 用途 |
|-------|------|
| 上级通知/文件要求学习的具体精神 | 确保学习体会准确对应 |
| 参学人员发言提纲或发言要点 | 增强体会的真实性 |
| 本单位近期相关工作进展 | 使贯彻落实措施更具针对性 |

## 信息不足时的处理

- **无具体学习材料内容**：体会段落生成通用框架，标注 `【待补：请提供学习材料内容以完善体会段落】`
- **无参学人员名单**：参学人员填写 `【待补】`
- **无贯彻落实具体方向**：措施部分生成通用表述，建议用户结合实际修改
