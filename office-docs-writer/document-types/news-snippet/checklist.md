# 新闻稿校验清单

## 一、结构完整性

- [ ] 标题准确概括活动（机构+动词+主题）
- [ ] 导语一句话交代时间、机构、活动
- [ ] 活动内容描述完整
- [ ] 亮点或成效段落有实质内容

## 二、内容准确性

- [ ] 时间、地点、参与人员等基本信息准确
- [ ] 活动内容描述与信息源一致
- [ ] 数据（如参与人数）有依据

## 三、文风规范

- [ ] 语言简洁，无冗长表述
- [ ] 用词正面、积极，符合对外宣传风格
- [ ] 避免过度套话，突出实际做法

## 四、格式规范

- [ ] 落款格式正确（供稿单位）
- [ ] 段落清晰，无重复信息

## 五、信息缺口

- [ ] "待补"标记均已处理
