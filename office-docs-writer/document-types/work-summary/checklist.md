# 工作总结校验清单

## 一、结构完整性

- [ ] 包含"总体情况"段落（简明扼要）
- [ ] 包含"主要工作进展"段落（分项展开）
- [ ] 包含"经验做法"段落（有实质内容）
- [ ] 包含"存在的问题和原因"段落（客观真实）
- [ ] 包含"下一步工作打算"段落（具体可执行）

## 二、内容准确性

- [ ] 工作成效与信息源材料一致
- [ ] 数据准确，无随意四舍五入或估算
- [ ] 问题分析客观，不回避，不夸大
- [ ] 无凭空编造的工作成果

## 三、政治表述

- [ ] 总体情况段落中的政治表述准确、适当
- [ ] 无过度政治化表述（不该拔高的内容不拔高）

## 四、格式规范

- [ ] 总体情况段落简洁（不超过300字）
- [ ] 各工作领域分项清晰
- [ ] 编号层级一致（一级"一、"，二级"（一）"）
- [ ] 落款格式符合规范

## 五、信息缺口

- [ ] 所有"待补"标记均已处理或说明
