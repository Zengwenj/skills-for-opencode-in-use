## 示例一：从零开始撰写季度会议记录

### 场景描述

撰写[年份]年第一季度的党风廉政建设工作专题会记录，第一次使用这个 agent。

### 准备工作

1. 已下载相关学习材料到本地知识库
2. 已有上季度（[年份]年Q4）的会议记录
3. 已有本季度工作台账

### 操作步骤

#### 步骤1：提供基本信息

**用户输入：**
```
请帮我撰写[年份]年第一季度党风廉政建设工作专题会记录

会议基本信息：
- 会议时间：[年份]年3月25日 14:00-17:00
- 会议地点：[会议地点]
- 主持人：[姓名]，[部门名称]总经理
- 参会人员：[姓名列表]
- 记录人：[姓名]
```

**Agent响应：**
```
接下来需要以下信息：

1. 上季度会议记录文件路径（用于对照工作部署）
2. 本季度工作完成情况
3. 本季度学习材料
4. 本季度工作安排

请提供第1项：上季度会议记录文件路径
```

---

#### 步骤2：提供上季度会议记录

**用户输入：**
```
上季度会议记录：[知识库根目录]\会议记录\[年份]Q4党风廉政专题会.md
```

**Agent响应：**
```
读取上季度会议记录后，提取到以下工作部署：

党建工作（5项）：
1. 组织党员集中学习4次
2. 开展主题党日活动3次
3. 发展预备党员1名
4. 组织党员志愿服务2次
5. 完成党员积分管理

党风廉政建设工作（4项）：
1. 开展廉政谈话20人次
2. 组织警示教育2次
3. 完成廉政风险排查
4. 建立廉政档案

意识形态工作（3项）：
1. 开展意识形态专题学习1次
2. 组织员工思想动态调研
3. 加强网络意识形态管理

请提供本季度完成情况，可以：
1. 提供工作台账文件路径
2. 直接描述完成情况
```

---

#### 步骤3：提供本季度完成情况

**用户输入：**
```
本季度工作台账：[知识库根目录]\工作台账\[年份]Q1工作台账.xlsx

额外完成的工作：
1. 组织开展"学雷锋"主题党日活动
2. 参加上级党委组织的党建知识竞赛，获得团体二等奖
3. 完成2名入党积极分子培养考察
```

**Agent响应（示例格式，实际内容以材料为准）：**

```
读取工作台账后，生成工作完成情况对照表：

【党建工作】
| 部署内容 | 完成情况 | 完成时间 |

| 组织党员集中学习4次 | 已完成4次 | [日期] |
| 开展主题党日活动3次 | 完成2次 | [日期] |
| 发展预备党员1名 | 未完成 | - |
| ... | ... | ... |

未完成说明：
- 主题党日活动：[原因]，延至下季度
- 发展预备党员：[原因]，延至下季度

请提供学习材料
```

---

#### 步骤4：提供学习材料（方式一：本地文件）

**用户输入：**
```

1. 领导人重要讲话：
   [知识库根目录]\党建\重要讲话\[年份]\Q1\

   - [市级机构]：[知识库根目录]\党建\重要会议精神\[市级机构]\[年份]\Q1\
   - [机构名称]：[知识库根目录]\党建\重要会议精神\[机构名称]\[年份]\Q1\

3. 规纪法：
   [知识库根目录]\党建\规纪法\

4. 警示案例：
   [知识库根目录]\党建\警示案例\[年份]\Q1\

5. 行内制度：
   [知识库根目录]\行内制度\

6. 重要发文：
   [知识库根目录]\行内发文\[年份]\Q1\
```

**Agent响应：**
```

2. 客户服务维护（责任人：全员，时限：日常）

1. 配合风险管理部完成风险评估（牵头：风险管理部，配合：[部门名称]，责任人：[姓名]，时限：4月30日）

```

```
[生成完整的会议记录文档]
```

## 常见问题处理

### Q1: 文件路径错误怎么办？

**错误示例：**
```
文件不存在：[知识库根目录]\[文件路径]
```

**解决方法：**
1. 检查路径是否正确
2. 确认文件是否存在
3. 检查文件名拼写
4. 提供文件内容的文字摘要作为替代

---

### Q2: 网站获取失败怎么办？

**错误示例：**
```
无法访问网站：[网址]
```

**解决方法：**
1. 检查网址是否正确
2. 检查网络连接
3. 手动复制网页内容粘贴提供
4. 使用其他来源

---

### Q3: 内容太多如何精简？

**用户输入：**
```
学习材料太多，请精简处理，每个板块只保留最重要的3项内容
```

**Agent响应：**
```
精简处理后，每个板块保留3项：

【重要讲话】
1. [最重要1]
2. [最重要2]
3. [最重要3]

【重要会议精神】
...
```

---

### Q4: 如何调整格式？

**用户输入：**
```
请将会议记录调整为更简洁的格式：
1. 学习内容只用表格，不需要详细摘要
2. 工作安排合并为一页
3. 去除"学习讨论"和"贯彻落实要求"部分
```

**Agent响应：**
```
[生成简洁版会议记录]
```

## 最佳实践

### 1. 提前准备

- 定期更新本地知识库
- 建立规范的文件目录结构
- 及时下载重要文件到本地

### 2. 分步进行

- 先提供基本信息
- 再提供文件路径
- 最后提供补充说明

### 3. 灵活使用

- 可以生成完整记录
- 也可以只生成部分内容
- 根据需要调整格式

### 4. 持续优化

- 保存生成的记录作为参考
- 积累优质素材
- 不断改进工作流程
