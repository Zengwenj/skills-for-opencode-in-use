# 政策一致性清单

核查文档内容与政策要求的一致性：

## 一、政策表述准确性

- [ ] 引用的政策文件、会议精神表述准确
- [ ] 政策要求传达无遗漏、无曲解
- [ ] 政策术语使用规范（如"主体责任"表述正确）

## 二、工作部署对应

- [ ] 文中工作安排与上级部署要求对应
- [ ] 未遗漏上级重点工作要求
- [ ] 时间节点符合上级部署的时间要求

## 三、考核要求对应

- [ ] 工作内容覆盖考核细则中的主要考核项
- [ ] 对考核指标有明确的完成情况说明
- [ ] 未有意规避考核薄弱项

## 四、合规性

- [ ] 文档内容无违反现行政策法规之处
- [ ] 所述工作措施在政策允许范围内
- [ ] 无政策过时引用（已废止的文件/规定）
