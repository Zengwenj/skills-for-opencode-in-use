# 敏感内容清单

在输出文档前，核查以下敏感内容风险：

## 一、人员信息

- [ ] 文中人员信息（姓名、职务）已按用户要求处理（保留/脱敏/省略）
- [ ] 无未经授权的个人信息披露
- [ ] 无对具体人员的负面评价表述

## 二、机构信息

- [ ] 涉及其他机构的信息在对外文档中已按需处理
- [ ] 无泄露机构内部业务数据的风险（如用于对外发布场景）

## 三、政治敏感表述

- [ ] 文中无不当的政治表述或立场偏差
- [ ] 对重大事件的表述与官方口径一致
- [ ] 无可能产生歧义的政治性措辞

## 四、业务敏感信息

- [ ] 文中数据不含不应对外披露的业务信息
- [ ] 客户信息、财务数据等在适当场景下已脱敏或省略
- [ ] 合规风险、监管处罚等敏感事项表述恰当

## 五、内外部分发场景

- [ ] 已确认本文档的分发范围（内部/对外）
- [ ] 内容深度与分发范围相匹配（对外文档不含内部敏感细节）
