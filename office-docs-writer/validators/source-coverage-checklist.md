# 信息源覆盖清单

核查文档内容是否充分利用了用户提供的信息源：

## 一、信息源利用情况

- [ ] 用户提供的所有材料均已参考
- [ ] 关键信息（数据、部署要求、进展）已体现在文档对应段落
- [ ] 无重要信息被遗漏

## 二、信息缺口处理

- [ ] 信息不足之处已用 `【待补：...】` 明确标注
- [ ] 标注内容说明了需要什么信息（非仅写"待补"）
- [ ] 无以通用表述代替本应具体填写的内容

## 三、来源一致性

- [ ] 文档内容与信息源材料无冲突
- [ ] 如多源信息存在矛盾，已告知用户确认
- [ ] 未超出信息源范围进行推断或补充

## 四、覆盖完整性（按文档类型）

**专题会议记录**：
- [ ] 上阶段工作完成情况已覆盖考核通知中的主要工作项
- [ ] 学习传达内容有对应材料来源

**工作总结**：
- [ ] 主要工作进展覆盖了工作部署的各重点方向
- [ ] 经验做法有对应工作实例支撑
