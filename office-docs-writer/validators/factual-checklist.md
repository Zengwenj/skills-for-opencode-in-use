# 事实核查清单

对生成的文档逐项核查事实准确性：

## 一、数据准确性

- [ ] 文中所有数字均有来源材料依据
- [ ] 完成率、数量等数据与信息源材料一致
- [ ] 时间节点（日期、周期）与材料一致
- [ ] 人员名称、职务表述准确

## 二、事件与工作内容

- [ ] 工作内容描述与实际情况一致，无夸大
- [ ] 所述完成事项确实已完成（非计划中事项）
- [ ] 典型案例或做法真实，非虚构
- [ ] 未将他部门/他单位的成果归属本单位

## 三、引用与来源

- [ ] 引用上级文件时，文件名称准确
- [ ] 引用政策规定时，条款内容准确
- [ ] 无无来源的统计数据或结论

## 四、一致性检查

- [ ] 同一数字/表述在文档不同段落中一致
- [ ] 文档内部逻辑自洽，无自相矛盾
