# 政治表述规范

## 使用原则

1. 政治表述以**当前有效的重要会议和政策精神**为准
2. 不固定引用特定届次（如"二十大精神"），而是在收到用户提供的材料后，准确提炼当前相关表述
3. 如用户提供了具体会议文件或通知，以该文件为权威依据
4. 无材料依据时，不推测或凭空填写政治表述内容

## 核心政治表述库

以下为常用规范表述，实际使用时应结合当前有效文件：

### 政治立场类

- 增强"四个意识"（政治意识、大局意识、核心意识、看齐意识）
- 坚定"四个自信"（道路自信、理论自信、制度自信、文化自信）
- 做到"两个维护"（坚决维护党中央权威和集中统一领导，坚决维护习近平总书记核心地位）

### 工作要求类

常见规范表述（需与当前政策文件核对）：

| 使用场景 | 参考表述 |
|---------|---------|
| 学习贯彻 | 深入学习贯彻[当前重要会议精神] |
| 工作部署 | 按照[上级重要部署]要求 |
| 落实责任 | 切实履行全面从严治党主体责任 |
| 风险防控 | 守住不发生系统性风险的底线 |
| 廉洁从业 | 严格执行廉洁从业相关规定 |

## 政治表述校验规则

1. **避免过时引用**：不使用已被新文件替代的会议/文件名称
2. **避免臆造**：政策精神必须有来源材料支撑
3. **避免不当简化**：政治概念不随意缩写或重新表述
4. **党政文件优先**：以上级党委、政府正式发布的文件为最高权威

## 敏感表述处理

- 涉及政治敏感内容，参照 validators/sensitive-content-checklist.md
- 不确定时，保留 `[当前重要会议精神]` 占位符，由用户确认后填入
