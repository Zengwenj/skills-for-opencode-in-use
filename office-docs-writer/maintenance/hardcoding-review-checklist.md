# 硬编码自查清单

每次编辑 skill 文件后，运行本清单检查是否引入新的硬编码内容。

---

## 检查项

### 机构名称

- [ ] 无 `成都银行`、`成都锦行`、`成都锦行股份有限公司` 等具体机构名
- [ ] 无 `支行`、`总行`、`分行`、`支行党支部`、`总行党委` 等具体职级名称
- [ ] 已替换为 `[机构名称]`、`[上级机构]`、`[部门名称]` 等占位符

### 人名与职务

- [ ] 无具体人名（如 `张三`、`李四` 等示例外的真实姓名）
- [ ] 无具体职务名称与姓名的绑定（如 `张某某总经理`）

### 年份与日期

- [ ] 无固定年份（如 `2024`、`2025`）
- [ ] 已替换为 `[年份]`、`[下一年份]`、`[日期]` 等占位符
- [ ] 无"今年"、"去年"等相对时间表述（除非在抽象说明中）

### 文件路径

- [ ] 无 `D:\`、`C:\` 等绝对路径
- [ ] 已替换为 `[知识库根目录]` 等占位符

### 地名

- [ ] 无 `四川省`、`成都市`、`成都市`等具体地名
- [ ] 已替换为 `[省级机构]`、`[市级机构]` 等占位符

### 政治表述

- [ ] 无 `党的二十大精神`、`党的二十届X中全会精神` 等固定届次表述
- [ ] 已替换为 `当前有效的重要会议和政策精神` 或 `[当前重要会议精神]`

### 成功状态语言

- [ ] 无 `✅ 已读取`、`✅ 已生成`、`✅ 已保存` 等操作状态输出
- [ ] 无 `自动校准`、`自动应用`、`自动识别` 等过度承诺语言
